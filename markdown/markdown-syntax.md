##Markdown syntax guide

Elements | How-To 
-------- | --------
Headers | `#`,`##`,...,`######`
Emphasis | `*text*` italic,`**text**` bold
Lists (U) | `* Item 1` 
Lists (O) | `1. Item 1`
Links | `[label](link)`
Images | `![label](link)`
Blockquotes | `> text`
Inline code | `` `code` ``

For GitHub Flavored Markdown, click [here](https://guides.github.com/features/mastering-markdown/#GitHub-flavored-markdown).
